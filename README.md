# Website for KatyBooProductions
Written in Sveltekit, published through GitHub pages

# To rebuild/publish:
### Build pages
```
npm run build
```
### Ensure that CNAME file was not modified
- Check in docs folder

### Publish to Github
- Add all modified files
- Commit and push
